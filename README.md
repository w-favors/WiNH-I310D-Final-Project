### Project Summary

The goal of this project was to analyze consumer shopping behavior and identify the factors that influence whether individuals prefer online shopping, in-store shopping, or a mix of both. Within the python notebook, we performed exploratory analysis, constructed an ML model, and evaluated the model using a dataset we got from Kaggle. That dataset is also included here as a csv file.

### Instructions

To get this project running, download both the csv file and the python notebook and keep both in the same directory. Next, make sure you have python installed as well as some form of IDE that can interpret Jupyter Notebooks. The python libraries you will need to have installed are:
- pandas
- matplotlib
- sklearn
- seaborn

Once you have done all of the above and have the notebook in your IDE, simply run all.
